DeepSeek 本地部署介绍

什么是 DeepSeek？

DeepSeek（深度求索）是一家中国人工智能公司，专注于开发大型开源语言模型。它的模型具有强大的自然语言处理能力，在各个领域中取得了显著的成绩。

本地部署的优势

  • 更好的数据隐私保护：避免将敏感数据传输到外部服务器。
  • 更高的性能：避免网络延迟，提高处理速度。
  • 灵活性：根据需求自定义模型和训练参数。

DeepSeek 本地部署步骤

以下是 DeepSeek 本地部署的基本步骤：

  1. 环境准备：确保机器具有足够的硬件资源，如高性能 CPU/GPU 和足够的内存。
  2. 下载Ollama：AI模型平台
  3. 下载 DeepSeek 模型：从 下载模型和相关依赖。
  4. 配置WebUI：不必依赖于控制台
  5. 运行模型：在本地机器上运行 DeepSeek 模型，并进行测试和优化。

Ollama

为什么是Ollama？

Ollama 提供简洁的 API 接口，开发者可以通过这些接口轻松调用语言模型，进行自然语言处理、生成文本、情感分析等任务。

  1. 安装ollama ollama官网下载链接
  2. 在ollama 官网搜索deepseek-r1 ollama官网
  3. 命令行粘贴并运行命令，ollama run deepseek-r1:1.5b（根据电脑配置自主选择，建议4060不要超过8b,没有独显请谨慎下载)

可视化图文交互界面 Chatbox

虽然我们可以在本地正常使用 Deepseek 这个模型了，但是这个 AI 工具的面板是非常简陋的，很多人使用不习惯，这时我们就可以通过 Chatbox 这个可视化图文交互界面来使用它。

点击 Chatbox AI官网：https://chatboxai.app/zh，Chatbox 虽然有本地客户端，但我们也可以直接使用网页版。 连接 本地模型，确保 Ollama 服务已运行。

在 ChatBox 设置中选择 ollama api，填写：
API Endpoint: http://localhost:11434
API Key: 留空
Model Name: deepseek-r1（与 Ollama 模型名称一致）
也可使用AnythingLLM,LM Studio等软件

注意事项

检查ollama下载 命令行输入ollama -v 浏览器访问http://localhost:11434/验证

远程访问，同一个局域网： Environment="OLLAMA_HOST=0.0.0.0" Environment="OLLAMA_ORIGINS=*"
如果你有公网IP或内网穿透，就可以远距离访问你的本地模型

如何打开命令行（Windows 用户按 Win + R，输入 cmd）
